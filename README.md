# 522-rpi-bluetooth-dotnetcore

Exemple de création d'un serveur Bluetooth BLE sur Raspberry PI avec dotNet Core et le package DotnetBleServer

## Prérequis
* Raspberry PI 3 ou +
* dotNet Core 3.1
* le package DotnetBleServer disponible sur Nuget [https://www.nuget.org/packages/BluetoothLE.Server.Linux](https://www.nuget.org/packages/BluetoothLE.Server.Linux)
